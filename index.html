Sass ToolKit

guide d'installation

Étape 1 : Installation de Node.js et Sass

Pour commencer, assurez-vous que vous avez Node.js installé sur votre machine. Ouvrez un terminal ou un invite de commande et vérifiez la version de Node.js en tapant :

node -v

Ensuite, installez Sass en tapant :

npm install -g sass

Vérifiez que Sass est bien installé en tapant :

sass --version

Étape 2 : Création de la structure du projet

Créez un dossier nommé "sass" à la racine de votre projet. Dans ce dossier, ajoutez un fichier nommé "main.scss" pour écrire vos styles Sass. Créez également un fichier "index.html" à la racine du projet pour créer la page web. Vous pouvez ajouter un fichier "styles.css" dans un dossier "public" pour stocker le CSS compilé.

Étape 3 : Configuration de l'environnement de développement

Ouvrez Visual Studio Code et installez l'extension "Live Sass Compiler". Cette extension vous permettra de compiler automatiquement vos fichiers Sass en CSS.

  1. Ouvrez le dossier "sass" dans Visual Studio Code.

  2. Ouvrez un terminal dans Visual Studio Code.

  3. Initialisez un fichier "package.json" pour gérer les dépendances du projet en tapant :

    npm init

    Remplissez les informations nécessaires ou appuyez simplement sur "Entrée" pour utiliser les valeurs par défaut.

  4. Installez Sass en tant que dépendance de développement en tapant :

    npm install sass --save-dev

  5. Modifiez le fichier "package.json" pour ajouter un script "watch:sass" :

    "scripts": {
  "watch:sass": "sass --watch sass/main.scss public/styles.css"
}

    Ce script permettra de surveiller les changements dans le fichier "main.scss" et de compiler automatiquement en CSS dans le fichier "styles.css".

  6. Pour lancer la compilation de Sass, utilisez la commande :

    npm run watch:sass

    Vous verrez les messages de compilation dans le terminal chaque fois que vous enregistrerez des modifications dans vos fichiers Sass. Votre fichier CSS compilé sera automatiquement mis à jour.
